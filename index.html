内容

  • Web Storage API を使ったサンプルです。
  • 別のタブやウィンドウでこのページを開いて、保持している値を比べて下さい。

Web Storage には、2種類あります。

sessionStorage
ウィンドウ、タブ単位でのみ有効なストレージ。
ページのセッション中のみデータが保存されます。
localStorage
ウィンドウ、タブ単位に限定されないストレージ。
ブラウザを閉じたり再び開いたりしても持続します。
